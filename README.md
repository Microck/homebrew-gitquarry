# homebrew-gitquarry

Homebrew tap for the [GitQuarry](https://github.com/Microck/gitquarry) CLI - AI-powered GitHub repository search.

## Install

```bash
# Add the tap and install gitquarry
brew tap Microck/gitquarry
brew install gitquarry
```

## Upgrade

```bash
# Update Homebrew and upgrade gitquarry
brew update
brew upgrade gitquarry
```

## Packages

| Package   | Description                    |
|-----------|--------------------------------|
| gitquarry | AI-powered GitHub search tool |

## Requirements

- [Homebrew](https://brew.sh) - macOS/Linux package manager
- GitHub token (for API rate limits)
